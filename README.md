##food-delivery-dashboard
Food Delivery Excel Dashboard analyzing 231 orders across Mumbai, Delhi, Bangalore, Hyderabad, Pune &amp; Nagpur. Covers data cleaning, KPIs, delivery time, payment modes &amp; order status. Restaurants include Pizza Corner, Urban Bites, Spice Hub &amp; more.
🍕 Food Delivery Excel Dashboard

An Excel-based data analysis and dashboard project built on a real-world food delivery dataset.

![Dashboard](food%20delivery%20dashboard%20ss.png)

📊 What's Inside

- **Raw Data Cleaning** – Messy order data cleaned and structured for analysis
- **Cleaned Dataset** – 231 food delivery orders across cities like Mumbai, Delhi, Bangalore, Hyderabad, Pune, and Nagpur
- **Dashboard Sheet** – Summary metrics and KPIs for quick insights

📁 Dataset Overview

| Column         | Description                          |
|----------------|--------------------------------------|
| Order_ID       | Unique order identifier              |
| Customer_Name  | Name of the customer                 |
| City           | Delivery city                        |
| Restaurant     | Restaurant name                      |
| Category       | Food category (Pizza, Chinese, etc.) |
| Order_Amount   | Total order value (₹)               |
| Quantity       | Number of items ordered              |
| Customer_Rating| Rating given by customer (1–5)       |
| Payment_Mode   | UPI / Card / Cash / Wallet           |
| Order_Status   | Delivered / Cancelled / Pending      |
| Delivery_Time  | Time taken for delivery (minutes)    |
